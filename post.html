Detection of Adulterants in Milk

This project consists of ammonia gas sensor that is mounted on the top of the beaker which contains milk.The gas sensor is interfaced with the aurdino.The LCD is connected to the digital pin.

The sensor detects and gives the information to aurdino and displays the values in LCD.To heat and stir the milk the beaker is placed on the magnetic stirrer.And the computer is connected to the aurdino to upload the program.


The primary sensing will be done by using the ammonia gas sensor(Q135).The ammonia gas sensor detects the ammonia gas that was escaping from the gas while heating about 70 degree celcius.The sensor will read the value and and it give to the aurdino.
The analog signal that was getting from the sesor will be converted to the digital valuesby using the aurdino board.The voltage value of the aurdino can be converted to any units by means of programming.The values can be converted to ppm by programming the aurdino.

The converted signal will be displayed as a required value by using the LCD

Technologies&Components:
  • - Python
  • - IOT
  • - Aurdino
  • - Ammonia Gas Sensor-(MQ135)
  • - PhMeter & LCD display
